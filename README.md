BUILD DOCKER
===
```
./build-docker.sh <target>
```
```
./build-docker.sh pocketbook6
```

BUILD DOCKER (WITH SDK BUILDING)
===
```
./build-docker-sdk.sh <target>
```
```
./build-docker-sdk.sh pocketbook6
```



BUILD
===
```
./build.sh <target> src/<dir>
```
```
./build.sh pocketbook6 src/hello
```

BASH
===
```
./bash.sh <target>
```
```
./bash.sh pocketbook6
```
